# E-Commerce-System-Dashboard
This project presents an interactive payment analytics dashboard built in Microsoft Power BI to analyze transaction data from an e-commerce platform. The dashboard provides insights into payment methods, total sales and payment success rates at cities to help businesses monitor financial performance and customer payment behavior.

🚀 Project Overview

The dashboard transforms raw e-commerce payment data into meaningful insights using data cleaning, transformation, and visualization techniques. It helps stakeholders quickly identify trends, monitor payment performance, and make data-driven decisions.

📌 Key Features

Total Transaction Amount KPI

Payment Method Distribution (Credit Card, Debit Card, UPI, Cash on Delivery)

Top Performing Payment Methods

Top Performing Payment Methods at Cities

Interactive categoty slicer for dynamic analysis
